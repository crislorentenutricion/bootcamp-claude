Antes del bootcamp

Preparación previa

Completa esto antes del día para aprovechar al máximo las 5 horas

Crear cuenta en Claude (claude.ai)
Necesitas una cuenta de pago: Pro (20$/mes) o Team. El plan gratuito no incluye Claude Code ni acceso a MCPs.
⚠ Obligatorio
Instalar Claude Code (CLI)
En la terminal: npm install -g @anthropic-ai/claude-code y luego claude para iniciar sesión. Necesitas Node.js 18+ instalado.
⚠ Obligatorio
Tener acceso a tus herramientas
Verifica que puedes acceder a: Google Drive, Gmail o el email que uses con clientes, y cualquier herramienta donde tengas datos de tu negocio (Notion, Sheets, etc.).
⚠ Obligatorio
Hacer la lista de tus 5 procesos más repetitivos
Piensa en las 5 tareas que haces cada semana de forma idéntica o muy similar. Estos serán los candidatos a convertirse en skills automatizadas. Anótalas en el apartado "Tu diseño" de esta página.
✓ Recomendado
Traer el portátil cargado (y el cargador)
Trabajaremos directamente en el terminal. Asegúrate de tener batería o enchufe disponible. 5 horas con la terminal abierta consume bastante.
↑ No te olvides
Leer los 4 pilares (sección más abajo)
No hace falta entenderlos a fondo, solo familiarizarte con los conceptos: CLAUDE.md, convenciones, skills y MCPs. Te ayudará a seguir el ritmo desde el principio.
↗ Opcional pero útil

03
Referencia

Los 4 pilares del sistema

Referencia rápida para usar durante el bootcamp

1 CLAUDE.md

El cerebro

El "contrato de trabajo" de tu asistente. Se carga en cada conversación. Define quién eres, qué haces, cómo te comunicas y qué reglas seguir siempre.

# Mi Negocio Titular: Ana García Marca: Ana García Estudio Servicio: Fotografía de bodas Tono: Cercano, elegante
2 Convenciones

La memoria

Los manuales detallados de cada proceso. Claude los consulta cuando va a ejecutar una tarea. Si cambias una regla o un precio, lo cambias aquí.

convenciones/ ├── clientes/ │ └── onboarding.md ├── servicios/ │ └── presupuesto.md └── comunicacion/ └── plantillas-email.md
3 Skills

Los flujos

Un proceso convertido en comando. Escribes "envía presupuesto a Ana" y Claude ejecuta los 6 pasos del proceso completo, con puntos de revisión.

--- name: enviar-presupuesto description: Triggers: "envía presupuesto a [nombre]" --- # Paso 1: leer datos del cliente # Paso 2: calcular precio # Paso 3: ESPERAR APROBACIÓN # Paso 4: generar PDF + email
4 MCPs

Las conexiones

Los "enchufes" que dan a Claude acceso real a tus herramientas. Sin MCP, Claude solo puede leer archivos locales. Con MCPs, puede tocar Drive, Gmail, Notion…

MCPs útiles: → Google Drive / Sheets → Gmail / Calendar → Notion → Canva → GitHub → Slack / WhatsApp (vía API)
→ Cómo fluye todo Escribes una orden en lenguaje natural → Claude lee el CLAUDE.md → activa la skill correcta → consulta las convenciones → usa los MCPs para acceder a tus herramientas → propone el resultado → tú apruebas → Claude ejecuta.
